# Task Manager App - AWS API

Una aplicación de lista de tareas (Todo List) construida con **React** y conectada a una infraestructura de **AWS (API Gateway, Lambda y DynamoDB)**.

## Estructura del Proyecto
* `/frontend`: Contiene el código fuente de la aplicación React.
* `/docs`: Documentación adicional, incluyendo la colección de Postman para pruebas de la API.

## Cómo empezar
1. Clona este repositorio.
2. Ve a la carpeta del proyecto: `cd frontend`.
3. Instala las dependencias: `npm install`.
4. Ejecuta la aplicación en modo desarrollo: `npm run dev`.
